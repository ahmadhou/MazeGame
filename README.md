🎮 وصف المشروع

هذا المشروع عبارة عن لعبة ثلاثية الأبعاد (3D) تم تطويرها باستخدام OpenGL الحديث (Core Profile 3.3) ولغة ++C.
تدور فكرة اللعبة حول متاهة (Maze) يتحكم فيها اللاعب من منظور الشخص الأول (First Person)، حيث يقوم بالحركة داخل البيئة، جمع العناصر، والوصول إلى نقطة النهاية ضمن وقت محدد.

تم بناء المشهد بالكامل باستخدام OpenGL من الصفر، بما في ذلك الأرضية، الجدران، والعناصر التفاعلية، مع استخدام الخامات (Textures) والإضاءة الأساسية عبر Shaders لإعطاء مظهر بصري مناسب.

🕹 طريقة اللعب
يتحكم اللاعب بالحركة باستخدام لوحة المفاتيح (W, A, S, D) والماوس للتحكم بالكاميرا.
توجد عناصر قابلة للجمع موزعة داخل المتاهة.
الهدف هو جمع جميع العناصر ثم الوصول إلى منطقة النهاية.
يفوز اللاعب عند جمع كل العناصر والوصول للنهاية.
يخسر اللاعب في حال انتهاء الوقت المحدد.
⚙️ التفاصيل التقنية
تم استخدام التقنيات التالية:
OpenGL (Core Profile 3.3)
GLFW لإدارة النافذة والإدخال
GLEW للتعامل مع OpenGL Extensions
GLM للعمليات الرياضية والتحويلات
stb_image لتحميل الصور (Textures)
أهم الميزات:
نظام كاميرا من منظور الشخص الأول (FPS Camera)
استخدام مصفوفات Model / View / Projection
تطبيق الخامات والإضاءة عبر Shaders
نظام تصادم باستخدام AABB
حلقة لعبة (Game Loop) تعمل بشكل مستمر (Update + Render)
تحميل الملفات باستخدام مسارات نسبية (Relative Paths)
